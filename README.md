# 🏗️ Worker Check-In System

## 🚀 Getting Started

### 🛠️ Prerequisites

- Node.js >= 18
- pnpm

### 🔐 Environment Variables

```bash
DATABASE_URL=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
NODE_ENV=
```

### 📦 Install & Run Locally

```bash
# Install dependencies
pnpm install

# Run local development server
pnpm dev
```
